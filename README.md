# 2017-18 Arrests Civil Rights Data Collection

***👥 About Dataset***

[Dataset](https://catalog.data.gov/dataset/2017-18-arrests-civil-rights-data-collection) from data.gov.
'This set of Excel file contains data on student referrals to law enforcement by disability and student-related arrests by disability for all states. It also contains data on incidents of offenses by type of offense for all states.' (OCR)

***❔ Research Questions***

**Part 1**
1. Which types of offenses are most common nationally?
2. Which states have the highest overall referral and arrest rates?
3. What proportion of referrals result in an arrest by state?

**Part 2**
1. Which states show the largest disparity in referral and arrest representation between students with and without disabilities?
